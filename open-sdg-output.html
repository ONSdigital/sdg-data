UK SDG data service

Open SDG output

This output includes a variety of endpoints designed to support the Open SDG platform. Examples are below:

Headlines

CSV/JSON files containing "headline" (aggregated only) data for indicators

  • en/headline/3-6-1.csv
  • en/headline/3-6-1.json
  • en/headline/3-2-2.csv
  • en/headline/3-2-2.json
  • en/headline/8-5-2.csv
  • en/headline/8-5-2.json
  • etc...

Data

CSV/JSON files containing fully disaggregated data for indicators

  • en/data/3-6-1.csv
  • en/data/3-6-1.json
  • en/data/3-2-2.csv
  • en/data/3-2-2.json
  • en/data/8-5-2.csv
  • en/data/8-5-2.json
  • etc...

Edges

CSV/JSON files containing "edges" (relationships between disaggregations) data for indicators

  • en/edges/3-6-1.csv
  • en/edges/3-6-1.json
  • en/edges/3-2-2.csv
  • en/edges/3-2-2.json
  • en/edges/8-5-2.csv
  • en/edges/8-5-2.json
  • etc...

Combined edges and data

JSON files containing both the fully-disaggregated and "edges" data mentioned above

  • en/comb/3-6-1.json
  • en/comb/3-2-2.json
  • en/comb/8-5-2.json
  • etc...

Metadata

JSON files containing metadata for the indicators

  • en/meta/3-6-1.json
  • en/meta/3-2-2.json
  • en/meta/8-5-2.json
  • etc...

Zip file of CSV data

Zip files containing all indicators in CSV form

  • en/zip/all_indicators.zip

Zip file information

JSON file containing information about the above-mentioned zip files

  • en/zip/all_indicators.json

Reporting status

JSON file containing information about the reporting status of the indicators

  • en/stats/reporting.json

Translations

JSON file containing all the translations used in the platform

  • en/translations/translations.json
