UK SDG data service

SDMX output

This output has an SDMX file for each indicator's data, plus one SDMX file with all indicator data. This data uses numbers and codes only, so is not specific to any language.The metadata is output in a 'meta' subfolder' and similarly has one per indicator plus an 'all' file. Translations of the metadata are included in each file using the 'lang' attribute. The data for this output uses the following data structure definition: https://registry.sdmx.org/ws/public/sdmxapi/rest/datastructure/IAEG-SDGs/SDG/latest/?format=sdmx-2.1&detail=full&references=children Examples are below:

  • sdmx-global/all.xml (all data)
  • sdmx-global/meta/all.xml (all metadata)
  • sdmx-global/14-7-1.xml (data by indicator)
  • sdmx-global/meta/14-7-1.xml (metadata by indicator)
  • sdmx-global/archived-17-5-1.xml (data by indicator)
  • sdmx-global/meta/archived-17-5-1.xml (metadata by indicator)
  • sdmx-global/1.xml (data by goal)
  • sdmx-global/meta/1.xml (metadata by goal)
  • sdmx-global/2.xml (data by goal)
  • sdmx-global/meta/2.xml (metadata by goal)
  • etc...
